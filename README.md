Memórias Manchadas é o repositório oficial do site de RPG de Mesa com temática de Investigação Paranormal e Terror Cósmico, fortemente inspirado no universo de Enigma do Medo.

Este site serve como um ponto de encontro e uma biblioteca de recursos para Mestres (ST) e Jogadores que se aventuram em campanhas onde a sanidade é seu recurso mais valioso e o desconhecido espreita em cada sombra.
